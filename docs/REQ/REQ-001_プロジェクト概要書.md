## 1. プロジェクト概要

### 1.1 システム名称

**personal-agent**（パーソナルエージェントシステム）

### 1.2 システム概要

日常的なタスク管理から作業の調整・依頼まで、様々なユースケースに対応するAIエージェントシステム。
チャット形式の自然言語インターフェースを通じて、Redmineと連携したタスク管理・スケジュール調整・情報収集・文書作成支援を行う個人向けAIアシスタント。

### 1.3 プロジェクト種別

- **種別**: 新規構築
- **利用形態**: 個人利用（シングルユーザー）
- **実行環境**: ローカルPC

---

## 2. 背景と目的

### 2.1 背景

現状、タスク管理・スケジュール調整・情報収集などの作業を手作業・アナログな方法で行っており、
以下の課題が存在する。

- タスクの登録・更新・確認に都度 Redmine の Web UI を操作する手間がかかる
- 複数の情報源から情報を収集・整理する作業が煩雑
- スケジュール確認と作業指示が分断されている

### 2.2 目的

AIエージェントによる自然言語インターフェースを提供し、上記の手作業・アナログ業務をデジタル化・自動化する。
ユーザーはチャット画面で指示するだけで、タスク管理・情報収集・文書作成などの作業をエージェントが代行する。

### 2.3 期待効果

- タスク管理の操作工数削減（Redmine 直接操作の排除）
- 情報収集・整理の自動化
- 作業依頼からタスク登録までのシームレス化

## 3. スコープ

### 3.1 対象範囲（フェーズ1 - 初期リリース）

| # | 機能カテゴリ | 説明 |
|---|-------------|------|
| 1 | Redmineタスク管理（CRUD） | タスクの作成・検索・更新・優先度調整 |
| 2 | チャットUI | 自然言語でエージェントに指示を出す画面 |
| 3 | タスク一覧ダッシュボード | Redmineタスクの状態を可視化する画面 |

### 3.2 対象範囲（フェーズ2 - 追加機能）

| # | 機能カテゴリ | 説明 |
|---|-------------|------|
| 4 | メモ・情報収集 | ウェブ検索・情報整理のエージェント支援 |
| 5 | 予定・カレンダー管理 | スケジュール確認・登録のエージェント支援 |
| 6 | 文書作成・編集支援 | レポート・メールの草案作成支援 |

### 3.3 対象外

- マルチユーザー対応（認証・権限管理）
- クラウドデプロイ・外部公開
- Redmine 以外のプロジェクト管理ツール連携（初期リリース時点）
