### UC-001: タスクを作成する

**概要**: ユーザーがチャットで指示を出し、エージェントがRedmineにタスクを登録する。

**アクター**: ユーザー

**事前条件**:
- Redmineが起動していること
- エージェントがRedmineに接続できること
- 対象Redmineプロジェクトが存在すること

**基本フロー**:
1. ユーザーがチャットにタスク作成の指示を入力する
2. エージェントが入力を解析し、タイトル・説明・優先度・期日を抽出する
3. エージェントがMCPを通じてRedmineにチケットを作成する
4. エージェントが作成完了メッセージ（チケットURL含む）をチャットに返す
5. ダッシュボードが更新され、新しいタスクが表示される

**代替フロー**:
- 2a. タイトルが不明な場合、エージェントが確認メッセージを返す
- 3a. Redmine接続失敗の場合、エージェントがエラーメッセージを返す

**事後条件**:
- Redmineに新しいチケットが登録されている

---

### UC-002: タスク一覧を確認する

**概要**: ユーザーがチャットで問い合わせ、エージェントがRedmineのタスク一覧を返す。

**アクター**: ユーザー

**事前条件**:
- Redmineが起動していること

**基本フロー**:
1. ユーザーがチャットにタスク一覧の問い合わせを入力する（例：「今日のタスクは？」）
2. エージェントが検索条件（ステータス・期日・キーワード等）を解析する
3. エージェントがMCPを通じてRedmineからチケット一覧を取得する
4. エージェントがタスク一覧を読みやすい形式（Markdown）でチャットに返す

**代替フロー**:
- 4a. 該当タスクが0件の場合、「該当するタスクはありません」と返す

**事後条件**:
- タスク一覧がチャットに表示されている

---

### UC-003: タスクのステータスを更新する

**概要**: ユーザーがチャットで指示し、エージェントがRedmineタスクのステータスを変更する。

**アクター**: ユーザー

**事前条件**:
- Redmineが起動していること
- 更新対象のタスクが存在すること

**基本フロー**:
1. ユーザーがチャットにステータス更新の指示を入力する（例：「#123を完了にして」）
2. エージェントが対象チケット番号とステータスを解析する
3. エージェントがMCPを通じてRedmineのチケットを更新する
4. エージェントが更新完了メッセージをチャットに返す

**代替フロー**:
- 2a. 対象チケットが曖昧な場合、エージェントが候補を提示して確認する

**事後条件**:
- Redmine チケットのステータスが更新されている

### UC-007: 優先タスクのレポートを受け取る

**概要**: エージェントが未完了タスクを分析し、優先度・期日に基づいた対応順序を提案する。

**アクター**: ユーザー

**基本フロー**:
1. ユーザーが「今日何をすればいい？」などの問い合わせをチャットに入力する
2. エージェントがRedmineから未完了タスク一覧を取得する
3. エージェントが期日・優先度を分析し、対応順序を決定する
4. エージェントが優先タスクリストとコメントをチャットに返す

### UC-008: エージェントとチャットする

**概要**: ユーザーがチャットUIを通じてエージェントと対話する基本操作。

**アクター**: ユーザー

**基本フロー**:
1. ユーザーがブラウザでチャット画面にアクセスする
2. ユーザーがテキストボックスにメッセージを入力し、Enterキーまたは送信ボタンで送信する
3. ローディングインジケータが表示される
4. エージェントが処理を実行する（必要に応じてツールを呼び出す）
5. エージェントの返答がチャット画面に表示される
6. 会話が継続される

**代替フロー**:
- 4a. エラーが発生した場合、エラーメッセージがチャットに表示される

---

### UC-009: ダッシュボードでタスク状況を確認する

**概要**: ユーザーがダッシュボード画面でRedmineタスクの状況を視覚的に確認する。

**アクター**: ユーザー

**基本フロー**:
1. ユーザーがブラウザでダッシュボード画面にアクセスする
2. ダッシュボードがRedmine APIからタスク一覧を取得する
3. タスクがステータス別にグルーピングされて表示される
4. 期日が近いタスクはハイライト表示される
5. ユーザーがタスクタイトルをクリックするとRedmineのチケット詳細ページに遷移する
